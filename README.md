
# Web Crawler

```
An web crawling application to scrap data from - 'https://www.millet.jp/'
```


# Used tools

```
# Python
# Selenium
# Pandas
# DotENV
```


# Commands to Build & Run

```
1. virtualenv -p /usr/bin/python3 venv
2. source venv/bin/activate
3. pip install -r requirements.txt 
4. python3 main.py
```
